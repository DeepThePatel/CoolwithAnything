Login Screen

User can enter their username and password to navigate to the "Home" screen by pressing the "Login" button. User can also sign up for an account by pressing the "Sign Up" button which navigates them to the Sign Up screen.

Sign Up Screen

User enters their information and clicks the "Sign-up" button to register for an account. The user is then taken to the "Home" page. User can also navigate back to login screen with the "Back to Login" button if they already have an existing account.

Forgot Password Screen

User will enter email and be given a link to reset password.User can also navigate back to login screen with the "Back to Login" button if they already have an existing account.

Home Screen

Home screen. User can see events they have for the current date. User can click on each event which navigates them to the "Edit Day" page. User can also click the calendar and the Current day is highlighted with a circle. User can also edit and delete and edit task and see how many active tasks are left

Add Screen

User is given the option to create either a task, meal, workout, or he can cancel.

Create Task Screen

User can enter inputs for a task and he can either create or cancel the task

Create Workout Screen

User can enter inputs for a Workout and he can either create or cancel the Workout

Create Meal Screen

User can enter inputs for a Meal and he can either create or cancel the Meal

Browse Workout (part 1)

Users can utilize the screen to browse various workouts and click on them to access more information.

Browse Workout (part 2)

When the user clicks on a specific workout, they are presented with this screen containing details and the option to save it to their list of saved workouts.

Saved Workouts

Users can browse through their saved workouts and can either edit, share, or delete them.

Browse Cookbook (part 1)

Users can utilize the screen to browse various meals and click on them to access more information.

Browse Cookbook (part 2)

When the user clicks on a specific workout, they are presented with this screen containing details and the option to save it to their list of saved workouts.

Saved Cookbook

Users can browse through their saved Meals and can either edit, share, or delete them.

Settings Screen (part 1)

User can view and edit all the personal information entered when creating an account.

Settings Screen (part 2)

user can check payment methods, transaction history, terms of service, QR code, and sign out.

Settings Screen (part 3)

user can switch to dark mode, change the language, access support, and view the current version they have.
